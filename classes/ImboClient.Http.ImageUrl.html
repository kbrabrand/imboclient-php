Arguments

$img

string

The identifier of the image to be used as a watermark. Can be omitted if the server is configured with a default watermark.

$width

int

The width of the watermark

$height

int

The height of the watermark

$position

string

The position of the watermark on the original image, 'top-left', 'top-right', 'bottom-left', 'bottom-right' or 'center'. Defaults to 'top-left'.

$x

int

Offset in the X-axis relative to the $position parameter. Defaults to 0
